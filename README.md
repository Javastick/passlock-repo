# PassLock Pro

Aplikasi penyimpanan kredensial login dengan enkripsi ganda (SHA-256 + AES), autentikasi biometrik simulasi, dan sistem pemulihan darurat via email.

## Teknologi
- Python 3.11
- Tkinter
- PyCryptodome
- smtplib

## Struktur Fitur
- Enkripsi password
- Simulasi biometrik
- Pemulihan darurat
